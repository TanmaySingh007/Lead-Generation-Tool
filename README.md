# Caprae Capital AI-Readiness Lead Generation Challenge

## Overview
This project demonstrates a smart lead generation tool for SaaS companies, built as a Jupyter Notebook. It loads, cleans, enriches, scores, and filters leads, providing an interactive and exportable workflow.

## Setup
1. Clone this repository.
2. Install requirements (see below).
3. Open `leadgen_ai_challenge.ipynb` in Jupyter Lab or Notebook.

## Requirements
- Python 3.8+
- pandas
- ipywidgets
- jupyter

Install with:
```bash
pip install pandas ipywidgets jupyter
```

## Usage
- Run all cells in the notebook.
- Use the interactive widgets to filter and export leads.

## Dataset
A sample SaaS company dataset (`sample_saas_companies.csv`) is included. You can replace it with your own CSV with similar columns.

## Deliverables
- Jupyter Notebook: `leadgen_ai_challenge.ipynb`
- 1-page report: `report.md`
- Sample dataset: `sample_saas_companies.csv`
